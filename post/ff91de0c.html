配置 SSH

  • 主目录下(/Users/用户名)

  • 如果不存在 .ssh 文件夹，新建

  • 切换文件目录

  • 生成公私钥对

  • 一路回车，出现如下画面表示成功

  • 查看公钥

  • 复制密钥，进入 github ，然后点击 Setting -> SSh and GPG keys -> New SSH key ，将复制 id_rsa.pub 内容粘贴上去即可

  • 验证是否成功
