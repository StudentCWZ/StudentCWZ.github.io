常用命令

  • brew 常用命令
命令 说明
安装软件 brew install xxx
卸载软件 brew uninstall xxx
搜索软件 brew search xxx
更新软件 brew upgrade xxx
查看列表 list
更新 brew brew update
清理所有包的旧版本 brew cleanup
清理指定包的旧版本 brew cleanup $FORMULA
查看可清理的旧版本包，不执行实际操作 brew cleanup -n
